# ChatGptPromptsMenu
Create floating menu containing the prompts for the session (for the question)

<ins>How to install as a bookmark (for pc's that can't install chrome extensions)</ins>
1. Copy the code from [`minimized.js`](https://github.com/snirex/ChatGptPromptsMenu/blob/main/minimized.js)
2. Press `ctrl+d` to create bookmark
3. Edit the bookmark and pastein it the text you've copied in step (1)
4. Name the bookmark and close the popup window.

Now you can press the bookmark while you're in chatgpt site to activate the menu
